# Como pedir correccion

Usa pedidos concretos. Asi la IA puede actuar como profesor y tecnico.

## Pedido minimo

```text
termine pieza NN.
Archivo: ...
Comando que corri: ...
Resultado: ...
Revisame como profesor/tecnico.
```

## Pedido ideal

```text
termine pieza NN.

Objetivo de la pieza:
Mi solucion esta en:
Comando que corri:
Que funciono:
Que no entiendo:
Caso que creo que puede fallar:

Revisame asi:
- primero bugs o riesgos
- despues mejoras chicas
- despues 3 preguntas para comprobar que entendi
```

## Para pedir ayuda sin recibir la solucion completa

```text
Estoy trabado en pieza NN.
No me des la solucion completa.
Dame una pista y una pregunta para destrabarme.
```

## Para pedir revision de proyecto

```text
Revisa esto como si fuera demo para una persona tecnica.
Quiero saber si se entiende, si es seguro y que falta para mostrarlo.
```

## Regla

Si todavia no ejecutaste o no explicaste que intentaste, primero hacemos eso.
La correccion mas util aparece cuando hay un intento real.
